CSS Selectors

CSS selectors define the pattern to select elements to which a set of CSS rules are then applied.

Syntax

Selector{property1: value1; property2: value2;}

Types Of Selectors

  1. Basic Selectors
    • Universal Selector
    • Element Selector
    • Class Selector
    • Id Selector
    • Attribute Selector
  2. Combinator Selectors
    • Descendant Selector
    • Child Selector
    • General Sibling Selector
    • Adjacent Sibling Selector
  3. Group Selectors
  4. Psuedo Selectors
    • Psuedo Class Selector
    • Psuedo Element Selector
